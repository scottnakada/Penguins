<h1>Penguins App</h1>

<h2>What does this app do</h2>
  <h4>
    <p>
      This is an educational app, with information about Penguins.
    </p>
  </h4>
  <h2>What can I learn about Penguins</h2>
    <h4>
      <ol>

        <li>How do they behave</li>
